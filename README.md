This project was developed as part of the Udhgam Hackathon, focusing on accessibility, multilingual support, and intelligent text processing.
It is a compact yet powerful AI tool that converts long text into a rich summary with emotion analysis, multi-language translation, dual audio output, and ASL sign-language representation.

Using Gradio, Transformers, gTTS, and deep-translator, the system can:

Generate clean summaries using BART

Detect emotional tone with an advanced NLP classifier

Translate into Hindi, French, Spanish, English, etc.

Produce audio in English + translated language

Display ASL signs using word GIFs and finger-spelling

Run smoothly on CPU/GPU (GPU recommended)
